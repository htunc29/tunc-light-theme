# Tunc Dark Theme

A clean, LinkedIn-inspired light theme for Visual Studio Code.

## Features

- Light theme with LinkedIn's signature blue (#0A66C2) accents
- Warm gray sidebar and panels (#F3F2EF)
- Blue status bar matching LinkedIn's brand
- Carefully crafted syntax highlighting for readability
- Supports all major languages: JavaScript, TypeScript, Python, HTML, CSS, JSON, YAML, Markdown and more

## Color Palette

| Element | Color |
|---------|-------|
| Background | `#ffffff` |
| Sidebar | `#f3f2ef` |
| Status Bar | `#0A66C2` |
| Keywords | `#0A66C2` |
| Functions | `#b24020` |
| Strings | `#057642` |
| Classes/Types | `#004182` |
| Numbers | `#8344a5` |
| Comments | `#999999` |

## Installation

1. Open **Extensions** sidebar in VS Code (`Ctrl+Shift+X`)
2. Search for `Tunc Light Theme`
3. Click **Install**
4. Go to **File > Preferences > Color Theme** and select **Tunc Light Theme**
