# Tunc Theme

A clean, LinkedIn-inspired theme pack for Visual Studio Code — includes both **Light** and **Dark** variants.

## Themes

### Tunc Light Theme
A bright, clean theme with LinkedIn's warm gray panels and signature blue accents.

### Tunc Dark Theme
A deep black theme with LinkedIn blue borders, accents and status bar.

## Installation

1. Open **Extensions** sidebar in VS Code (`Ctrl+Shift+X`)
2. Search for `Tunc Light Theme`
3. Click **Install**
4. Go to **File > Preferences > Color Theme** (`Ctrl+K Ctrl+T`)
5. Select **Tunc Light Theme** or **Tunc Dark Theme**
